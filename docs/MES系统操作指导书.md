# MES系统操作指导书

## 一、系统登录

### 1.1 管理员登录
**操作步骤：**
1. 访问系统登录页面
2. 输入管理员账号（默认：admin）和密码
3. 点击"登录"按钮进入系统

**说明：** 管理员拥有系统全部功能权限，可进行用户管理、角色配置等操作。

### 1.2 普通用户登录
**操作步骤：**
1. 输入分配的用户账号和密码
2. 点击"登录"按钮
3. 进入系统后根据角色权限使用相应功能

## 二、常规管理

### 2.1 系统管理

#### 2.1.1 菜单管理
**功能路径：** 常规管理 → 系统管理 → 菜单管理

**操作步骤：**
1. 点击"菜单管理"进入菜单列表（树形结构）
2. **新增菜单：** 点击"新增" → 选择上级菜单 → 填写菜单名称、菜单编码、路由地址、图标等 → 点击"保存"
3. **配置菜单：** 支持多级菜单配置，可设置菜单类型（菜单/按钮）

#### 2.1.2 基础数据配置平台
**功能路径：** 常规管理 → 系统管理 → 基础数据配置平台

**操作步骤：**
1. 进入基础数据配置平台
2. 配置系统基础参数和平台设置
3. 保存配置并生效

#### 2.1.3 基础数据维护
**功能路径：** 常规管理 → 系统管理 → 基础数据维护

**操作步骤：**
1. 进入基础数据维护页面
2. 维护系统运行所需的基础数据
3. 支持数据的增删改查操作

### 2.2 数据字典
**功能路径：** 常规管理 → 数据字典

**操作步骤：**
1. 点击"数据字典"进入字典列表页面
2. **新增字典类型：** 点击"新增类型" → 填写类型编码、类型名称 → 点击"保存"
3. **新增字典数据：** 选择字典类型 → 点击"新增" → 填写字典值、字典标签、排序号 → 点击"保存"

## 三、资源分配管理

### 3.1 用户管理
**功能路径：** 资源分配管理 → 用户管理

**操作步骤：**
1. 点击"用户管理"进入用户列表页面
2. **新增用户：** 点击"新增"按钮 → 填写用户名、密码、真实姓名、手机号、邮箱等基本信息 → 选择所属部门 → 点击"保存"
3. **编辑用户：** 在用户列表中找到目标用户 → 点击"编辑" → 修改信息 → 点击"保存"
4. **删除用户：** 勾选目标用户 → 点击"删除" → 确认删除
5. **分配角色：** 点击用户"分配角色"按钮 → 勾选相应角色 → 点击"确定"

### 3.2 角色管理
**功能路径：** 资源分配管理 → 角色管理

**操作步骤：**
1. 点击"角色管理"进入角色列表页面
2. **新增角色：** 点击"新增" → 填写角色名称、角色编码、备注 → 点击"保存"
3. **分配权限：** 点击角色"分配权限"按钮 → 在权限树中勾选所需权限 → 点击"确定"
4. **编辑/删除：** 在角色列表中操作对应按钮

### 3.3 部门管理
**功能路径：** 资源分配管理 → 部门管理

**操作步骤：**
1. 点击"部门管理"进入部门树形列表
2. **新增部门：** 点击"新增" → 选择上级部门 → 填写部门名称、部门编码、排序号 → 点击"保存"
3. **编辑/删除：** 在部门树中操作对应按钮

### 3.4 班组管理
**功能路径：** 资源分配管理 → 班组管理

**操作步骤：**
1. 点击"班组管理"进入班组列表页面
2. **新增班组：** 点击"新增" → 填写班组编码、班组名称、班组长、联系电话 → 点击"保存"
3. **添加组员：** 点击班组"管理组员" → 从人员列表中选择添加
4. **班次设置：** 配置班组的工作班次（白班/夜班等）

### 3.5 生产设备管理
**功能路径：** 资源分配管理 → 生产设备管理

**操作步骤：**
1. 点击"设备管理"进入设备列表页面
2. **新增设备：** 点击"新增" → 填写设备编号、设备名称、设备类型、所属工序单元等 → 点击"保存"
3. **设备状态管理：** 查看设备运行状态、维护记录等

### 3.6 设备编组管理
**功能路径：** 资源分配管理 → 设备编组管理

**操作步骤：**
1. 进入设备编组管理页面
2. **创建编组：** 点击"新增" → 填写编组名称、编组说明 → 点击"保存"
3. **设备分配：** 将设备添加到相应编组中
4. **编组调整：** 根据生产需要调整编组成员

### 3.7 加工单元管理
**功能路径：** 资源分配管理 → 加工单元管理

**操作步骤：**
1. 进入加工单元管理页面
2. **新增单元：** 点击"新增" → 填写单元编码、名称、类型等 → 点击"保存"
3. **资源配置：** 为加工单元分配设备和人员

## 四、生产管理

### 4.1 生产订单录入
**功能路径：** 生产管理 → 生产订单录入

**操作步骤：**
1. 点击"生产订单录入"进入订单列表页面
2. **创建订单：** 点击"新增" → 填写订单编号、客户名称、产品型号、数量、交货日期等 → 点击"保存"
3. **订单评审：** 对订单进行技术评审和产能评审
4. **订单下达：** 评审通过后点击"下达"将订单投入生产

### 4.2 生产作业派工
**功能路径：** 生产管理 → 生产作业派工

**操作步骤：**
1. 点击"生产作业派工"进入派工单列表页面
2. **创建派工单：** 点击"新增" → 选择生产订单 → 选择派工班组 → 填写计划开工时间、计划完工时间 → 点击"保存"
3. **派发工序：** 在派工单详情中点击"派发工序" → 选择工序 → 指定执行人员
4. **下达派工：** 点击"下达"按钮将派工单下发给执行人员

## 五、物料管理

### 5.1 物料信息定义
**功能路径：** 物料管理 → 物料信息定义

**操作步骤：**
1. 点击"物料信息"进入物料列表页面
2. **新增物料：** 点击"新增" → 填写物料编码、物料名称、物料规格、单位、物料分类等基本信息 → 点击"保存"
3. **查询物料：** 输入物料编码或名称关键字 → 点击"查询"
4. **编辑/删除：** 在物料列表中操作对应按钮

### 5.2 物料需求计划（MRP）
**功能路径：** 物料管理 → 物料需求计划

**操作步骤：**
1. 点击"MRP运算"进入MRP页面
2. **选择订单：** 勾选需要进行MRP运算的生产订单
3. **执行运算：** 点击"执行MRP"按钮 → 系统自动计算物料需求
4. **查看结果：** 查看物料需求清单，包括需求数量、需求时间、库存缺口等
5. **生成采购计划：** 根据MRP结果生成相应的采购申请

3. **配置工序：** 点击流程"配置工序" → 拖拽添加工序 → 设置工序顺序、工序参数
4. **绑定BOM：** 将工艺流程与相应的物料清单进行关联

### 7.5 工艺内容编制
**功能路径：** 产品数据中心 → 工艺内容编制

**操作步骤：**
1. 点击"工艺内容编制"进入内容列表页面

## 十、仓库管理

### 10.1 库房管理

### 11.7 统计分析

4. **导出报表：** 点击"导出"生成统计报表

### 11.8 质检跟踪追溯

4. **问题追溯：** 追溯质量问题的根源

2. **创建流程：** 点击"新建流程" → 填写流程名称、流程分类 → 点击"创建"
3. **拖拽节点：** 从左侧节点库拖拽所需节点到画布
4. **配置节点：** 点击节点 → 在右侧属性面板中配置节点属性（名称、类型、参数等）
5. **连接节点：** 鼠标拖拽从一个节点的锚点到另一个节点建立连线
6. **保存发布：** 点击"保存"按钮保存流程设计

**节点类型说明：**
- **开始节点：** 流程起点（每个流程只能有一个）
- **结束节点：** 流程终点（可设置多个）
- **工序节点：** 表示生产工序
- **判断节点：** 条件分支判断
- **数据节点：** 数据记录/获取
- **通知节点：** 消息通知

## 十三、常见操作说明

### 13.1 查询操作
- **精确查询：** 在查询框中输入完整关键字
- **模糊查询：** 输入部分关键字进行模糊匹配
- **组合查询：** 使用多个查询条件进行组合查询
- **重置查询：** 点击"重置"按钮清除查询条件

### 13.2 批量操作
- **全选：** 点击列表表头的复选框
- **多选：** 按住Ctrl键点击多个目标
- **批量删除：** 勾选多个目标 → 点击"批量删除" → 确认删除
- **批量审核：** 勾选多个目标 → 点击"批量审核" → 填写审核意见 → 确认

### 13.3 导出操作
1. 在列表页面完成数据筛选
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV/PDF）
4. 点击"确定"下载导出文件

### 13.4 打印操作
1. 在需要打印的页面点击"打印"按钮
2. 选择打印机
3. 设置打印份数和页面范围
4. 点击"打印"
