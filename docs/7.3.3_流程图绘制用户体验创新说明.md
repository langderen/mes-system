# 7.3.3 流程图绘制用户体验创新说明

## 1. 功能概述

本系统内置了直观易用的可视化流程图绘制工具，专门为制造执行系统（MES）的流程设计需求而定制。该工具支持用户通过拖拽方式快速构建工艺流程、审核流程等各种业务流程图，无需编写代码即可完成复杂流程的可视化设计。

## 2. 核心功能特性

### 2.1 三栏式界面布局

流程图设计器采用现代化的三栏式布局，提供清晰的工作区域划分：

- **左侧节点库**：展示可用的工序节点，支持拖拽添加
- **中间画布区**：提供大尺寸网格画布，支持节点拖拽、连线、编辑
- **右侧属性面板**：实时编辑选中节点的属性（标题、描述、类型、排序等）

### 2.2 直观的节点管理

#### 2.2.1 节点类型

系统内置多种节点类型，满足不同业务场景需求：
- **开始节点**：流程起点
- **任务节点**：普通执行任务
- **审核节点**：审批/审核环节
- **结束节点**：流程终点

#### 2.2.2 节点操作

- **拖拽添加**：从左侧节点库拖拽节点到画布
- **点击添加**：直接点击左侧节点即可添加
- **位置调整**：拖拽节点自由移动位置
- **属性编辑**：选中节点后在右侧面板编辑属性
- **删除节点**：点击节点上的删除按钮或按Delete键

### 2.3 智能连线机制

#### 2.3.1 锚点设计

每个节点都配有左右两个锚点（锚点）：
- **左侧锚点**（蓝色圆形）：作为连线终点
- **右侧锚点**（蓝色圆形）：作为连线起点

#### 2.3.2 连线方式

提供两种便捷的连线方式：

1. **锚点连线**：
   - 点击第一个节点的右侧锚点
   - 再点击第二个节点的左侧锚点
   - 自动生成平滑贝塞尔曲线连线

2. **属性面板连线**：
   - 选中起始节点
   - 点击右侧属性面板的「连到下一个」按钮
   - 再点击目标节点完成连线

#### 2.3.3 连线特性

- 自动生成平滑的贝塞尔曲线，视觉效果优雅
- 带箭头指示，明确流程方向
- 连线随节点位置自动调整
- 防止重复连线和自连接

### 2.4 网格画布系统

#### 2.4.1 视觉设计

- 浅色网格背景（28px间距），辅助对齐
- 大尺寸画布（1600×1000px），支持复杂流程
- 带滚动条，可轻松浏览大型流程
- 网格线采用低透明度设计，不干扰视觉

#### 2.4.2 智能定位

新节点自动避免重叠：
- 每5个节点自动换行
- 根据当前滚动位置智能计算添加位置
- 保持合理的节点间距

### 2.5 实时状态反馈

#### 2.5.1 保存状态指示器

- 顶部工具栏显示「未保存」状态标签
- 保存后变为绿色「保存成功」
- 添加节点、连线时显示相应状态提示

#### 2.5.2 节点选中反馈

- 选中节点有蓝色高亮边框
- 选中节点阴影加深
- 右侧属性面板显示选中节点信息

### 2.6 数据持久化

#### 2.6.1 存储结构

流程图数据采用JSON格式存储在 `sp_flow` 表的 `process` 字段中：

```json
{
  "nodes": [
    {
      "id": "node_xxx",
      "operId": "工序ID",
      "title": "节点标题",
      "desc": "节点描述",
      "type": "task",
      "sortNo": 1,
      "x": 100,
      "y": 120
    }
  ],
  "links": [
    {
      "from": "起始节点ID",
      "to": "目标节点ID"
    }
  ]
}
```

#### 2.6.2 一键保存

- 点击「保存流程图」按钮即可保存
- 保存时自动序列化当前画布状态
- 支持加载已有流程图继续编辑

## 3. 技术实现亮点

### 3.1 SVG渲染层

连线采用原生SVG渲染：
- 使用贝塞尔曲线（C指令）实现平滑连线
- 定义marker实现箭头效果
- 性能优异，支持大量节点和连线

### 3.2 拖拽交互

- 原生HTML5 Drag & Drop API
- 节点移动时实时更新连线
- 限制边界防止节点移出画布

### 3.3 分层设计

采用三层视觉架构：
1. **节点层**（z-index: 2）：可拖拽交互
2. **SVG连线层**（z-index: 10）：在节点之上绘制
3. **锚点层**（z-index: 3）：最上层，便于点击

### 3.4 状态管理

采用简单的状态对象管理：
```javascript
var state = {
    nodes: [],      // 节点数组
    links: [],      // 连线数组
    selectedId: null, // 当前选中节点
    drag: null,     // 拖拽状态
    connectFrom: null, // 连线起点
    seq: 1          // 节点序号
};
```

## 4. 用户体验优化

### 4.1 视觉设计

- **渐变背景**：节点头部采用蓝紫色渐变
- **圆角设计**：12px圆角，现代感强
- **阴影效果**：柔和的阴影，层次感强
- **悬停效果**：节点和锚点都有悬停反馈

### 4.2 操作便捷性

- **键盘快捷键**：Delete键删除选中节点
- **双击编辑**：双击节点提示编辑属性
- **一键清空**：确认后清空整个画布
- **撤销提示**：重要操作都有确认弹窗

### 4.3 引导式设计

- 空画布显示「从左侧拖入节点，开始设计流程图」提示
- 节点库有操作说明
- 属性面板有操作提示
- 连线过程有文字提示

## 5. 应用场景

### 5.1 工艺流程设计

- 设计产品加工工艺流程
- 配置各工序之间的流转关系
- 绑定实际工序数据

### 5.2 审核流程配置

- 设计多级审批流程
- 配置审核节点和流转条件
- 支持流程版本管理

### 5.3 业务流程建模

- 生产调度流程
- 物料转运流程
- 质量检验流程

## 6. 与现有系统集成

### 6.1 工序数据绑定

流程图节点可直接绑定系统中的工序（SpOper）数据，实现流程与实际业务的无缝衔接。

### 6.2 流程版本管理

支持流程版本控制，可保存多个版本的流程图设计。

### 6.3 脚本扩展

支持在流程中配置脚本内容（scriptContent），实现复杂的业务逻辑。

- 拖拽响应延迟<16ms
- 保存操作秒级完成
