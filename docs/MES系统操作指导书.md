# MES系统操作指导书

## 一、系统登录

### 1.1 管理员登录
**操作步骤：**
1. 访问系统登录页面
2. 输入管理员账号（默认：admin）和密码
3. 点击"登录"按钮进入系统

**说明：** 管理员拥有系统全部功能权限，可进行用户管理、角色配置等操作。

### 1.2 普通用户登录
**操作步骤：**
1. 输入分配的用户账号和密码
2. 点击"登录"按钮
3. 进入系统后根据角色权限使用相应功能

## 二、系统管理模块

### 2.1 用户管理
**功能路径：** 系统管理 → 用户管理

**操作步骤：**
1. 点击"用户管理"进入用户列表页面
2. **新增用户：** 点击"新增"按钮 → 填写用户名、密码、真实姓名、手机号、邮箱等基本信息 → 选择所属部门 → 点击"保存"
3. **编辑用户：** 在用户列表中找到目标用户 → 点击"编辑" → 修改信息 → 点击"保存"
4. **删除用户：** 勾选目标用户 → 点击"删除" → 确认删除
5. **分配角色：** 点击用户"分配角色"按钮 → 勾选相应角色 → 点击"确定"

### 2.2 角色管理
**功能路径：** 系统管理 → 角色管理

**操作步骤：**
1. 点击"角色管理"进入角色列表页面
2. **新增角色：** 点击"新增" → 填写角色名称、角色编码、备注 → 点击"保存"
3. **分配权限：** 点击角色"分配权限"按钮 → 在权限树中勾选所需权限 → 点击"确定"
4. **编辑/删除：** 在角色列表中操作对应按钮

### 2.3 菜单管理
**功能路径：** 系统管理 → 菜单管理

**操作步骤：**
1. 点击"菜单管理"进入菜单列表（树形结构）
2. **新增菜单：** 点击"新增" → 选择上级菜单 → 填写菜单名称、菜单编码、路由地址、图标等 → 点击"保存"
3. **配置菜单：** 支持多级菜单配置，可设置菜单类型（菜单/按钮）

### 2.4 部门管理
**功能路径：** 系统管理 → 部门管理

**操作步骤：**
1. 点击"部门管理"进入部门树形列表
2. **新增部门：** 点击"新增" → 选择上级部门 → 填写部门名称、部门编码、排序号 → 点击"保存"
3. **编辑/删除：** 在部门树中操作对应按钮

### 2.5 设备管理
**功能路径：** 系统管理 → 设备管理

**操作步骤：**
1. 点击"设备管理"进入设备列表页面
2. **新增设备：** 点击"新增" → 填写设备编号、设备名称、设备类型、所属工序单元等 → 点击"保存"
3. **设备分组：** 点击"设备分组" → 创建/管理设备组 → 将设备添加到相应分组

### 2.6 数据字典
**功能路径：** 系统管理 → 数据字典

**操作步骤：**
1. 点击"数据字典"进入字典列表页面
2. **新增字典类型：** 点击"新增类型" → 填写类型编码、类型名称 → 点击"保存"
3. **新增字典数据：** 选择字典类型 → 点击"新增" → 填写字典值、字典标签、排序号 → 点击"保存"

## 三、基础数据模块

### 3.1 物料信息管理
**功能路径：** 基础数据 → 物料信息

**操作步骤：**
1. 点击"物料信息"进入物料列表页面
2. **新增物料：** 点击"新增" → 填写物料编码、物料名称、物料规格、单位、物料分类等基本信息 → 点击"保存"
3. **查询物料：** 输入物料编码或名称关键字 → 点击"查询"
4. **编辑/删除：** 在物料列表中操作对应按钮

### 3.2 表格模板管理
**功能路径：** 基础数据 → 表格模板

**操作步骤：**
1. 点击"表格模板"进入模板列表页面
2. **新增模板：** 点击"新增" → 填写模板名称、模板编码 → 点击"保存"
3. **配置模板项：** 点击模板"配置项"按钮 → 新增表格列 → 设置列名称、列字段、列类型、是否必填等属性
4. **启用/禁用：** 操作对应按钮控制模板状态

## 四、生产数据模块

### 4.1 物料清单（BOM）管理
**功能路径：** 生产数据 → 物料清单

**操作步骤：**
1. 点击"物料清单"进入BOM列表页面
2. **新增BOM：** 点击"新增" → 选择产品（父级物料） → 填写版本、备注 → 点击"保存"
3. **添加BOM项：** 在BOM详情页点击"新增子项" → 选择子级物料 → 填写用量、损耗率、顺序等
4. **编辑/删除BOM项：** 在子项列表中操作对应按钮

### 4.2 工艺路线管理
**功能路径：** 生产数据 → 工艺路线

**操作步骤：**
1. 点击"工艺路线"进入路线列表页面
2. **新增路线：** 点击"新增" → 填写路线编码、路线名称、产品分类 → 点击"保存"
3. **配置工序：** 点击路线"配置工序" → 拖拽添加工序 → 设置工序顺序、工序参数
4. **绑定BOM：** 将工艺路线与相应的物料清单进行关联

### 4.3 工序管理
**功能路径：** 生产数据 → 工序管理

**操作步骤：**
1. 点击"工序管理"进入工序列表页面
2. **新增工序：** 点击"新增" → 填写工序编码、工序名称、工序类型、标准工时等 → 点击"保存"
3. **配置工艺内容：** 点击工序"配置工艺" → 添加工艺指导书、参数要求、检测标准等
4. **关联设备：** 设置工序所需设备和工具

### 4.4 工艺内容管理
**功能路径：** 生产数据 → 工艺内容

**操作步骤：**
1. 点击"工艺内容"进入内容列表页面
2. **新增内容：** 点击"新增" → 选择工序 → 填写内容类型、内容描述、附件等 → 点击"保存"
3. **内容类型支持：** 文本、图片、PDF文档、视频等

## 五、生产执行模块

### 5.1 生产派工
**功能路径：** 生产执行 → 生产派工

**操作步骤：**
1. 点击"生产派工"进入派工单列表页面
2. **创建派工单：** 点击"新增" → 选择生产订单 → 选择派工班组 → 填写计划开工时间、计划完工时间 → 点击"保存"
3. **派发工序：** 在派工单详情中点击"派发工序" → 选择工序 → 指定执行人员
4. **下达派工：** 点击"下达"按钮将派工单下发给执行人员

### 5.2 生产班组管理
**功能路径：** 生产执行 → 班组管理

**操作步骤：**
1. 点击"班组管理"进入班组列表页面
2. **新增班组：** 点击"新增" → 填写班组编码、班组名称、班组长、联系电话 → 点击"保存"
3. **添加组员：** 点击班组"管理组员" → 从人员列表中选择添加
4. **班次设置：** 配置班组的工作班次（白班/夜班等）

### 5.3 操作员管理
**功能路径：** 生产执行 → 操作员管理

**操作步骤：**
1. 点击"操作员管理"进入操作员列表页面
2. **新增操作员：** 点击"新增" → 填写操作员工号、姓名、所属班组、技能等级等 → 点击"保存"
3. **技能配置：** 设置操作员可执行的工序和设备

### 5.4 生产记录
**功能路径：** 生产执行 → 生产记录

**操作步骤：**
1. 点击"生产记录"进入记录列表页面
2. **查看记录：** 选择时间范围和派工单 → 点击"查询"查看生产执行记录
3. **记录内容：** 查看工序完成情况、产量、质量数据等

## 六、质量管理模块

### 6.1 质检计划管理
**功能路径：** 质量管理 → 质检计划

**操作步骤：**
1. 点击"质检计划"进入计划列表页面
2. **创建计划：** 点击"新增" → 填写计划名称、质检类型、抽检比例 → 选择关联的生产订单 → 点击"保存"
3. **配置检验项：** 在计划详情中添加检验项目和检验标准
4. **下达计划：** 点击"下达"将质检计划下发给质检人员

### 6.2 质检任务管理
**功能路径：** 质量管理 → 质检任务

**操作步骤：**
1. 点击"质检任务"进入任务列表页面
2. **接收任务：** 质检人员查看待执行任务
3. **执行质检：** 点击任务"执行" → 填写检验数据 → 上传检验照片 → 点击"提交"
4. **合格判定：** 根据检验结果判定产品是否合格

### 6.3 质检执行
**功能路径：** 质量管理 → 质检执行

**操作步骤：**
1. 点击"质检执行"进入执行页面
2. **扫描/输入产品：** 输入产品序列号或扫描条码
3. **逐项检验：** 按照检验项目逐一检测并记录数据
4. **提交结果：** 完成所有检验项后提交检验结果

### 6.4 质检数据统计
**功能路径：** 质量管理 → 质检统计

**操作步骤：**
1. 点击"质检统计"进入统计页面
2. **选择统计维度：** 按时间、产品、班组、工序等维度查看统计数据
3. **查看图表：** 查看合格率趋势图、不良项目分布图等
4. **导出报表：** 点击"导出"生成统计报表

### 6.5 不良品管理
**功能路径：** 质量管理 → 不良品管理

**操作步骤：**
1. 点击"不良品管理"进入不良品列表页面
2. **登记不良品：** 点击"新增" → 扫描/输入产品信息 → 选择不良现象 → 填写不良描述 → 点击"保存"
3. **不良品处理：** 选择处理方式（返工/报废/降级）并执行

## 七、仓库管理模块

### 7.1 仓库管理
**功能路径：** 仓库管理 → 仓库信息

**操作步骤：**
1. 点击"仓库信息"进入仓库列表页面
2. **新增仓库：** 点击"新增" → 填写仓库编码、仓库名称、仓库类型、地址等 → 点击"保存"
3. **库区划分：** 在仓库详情中配置库区（原材料区、成品区、不良品区等）

### 7.2 库位管理
**功能路径：** 仓库管理 → 库位管理

**操作步骤：**
1. 点击"库位管理"进入库位列表页面
2. **新增库位：** 点击"新增" → 选择仓库和库区 → 填写库位编码、库位名称 → 点击"保存"
3. **库位分配：** 设置库位的用途（存储、暂存、检验等）

### 7.3 入库管理
**功能路径：** 仓库管理 → 入库单

**操作步骤：**
1. 点击"入库单"进入入库列表页面
2. **创建入库单：** 点击"新增" → 选择入库类型（采购入库/生产入库/退货入库）→ 选择供应商/订单 → 点击"保存"
3. **添加入库明细：** 点击"添加物料" → 选择物料 → 填写数量、批次号、库位等
4. **确认入库：** 点击"确认入库"完成物料入库操作

## 八、工艺流程设计模块

### 8.1 流程图绘制
**功能路径：** 工艺设计 → 流程图设计

**操作步骤：**
1. 点击"流程图设计"进入设计页面
2. **创建流程：** 点击"新建流程" → 填写流程名称、流程分类 → 点击"创建"
3. **拖拽节点：** 从左侧节点库拖拽所需节点到画布
4. **配置节点：** 点击节点 → 在右侧属性面板中配置节点属性（名称、类型、参数等）
5. **连接节点：** 鼠标拖拽从一个节点的锚点到另一个节点建立连线
6. **保存发布：** 点击"保存"按钮保存流程设计

**节点类型说明：**
- **开始节点：** 流程起点（每个流程只能有一个）
- **结束节点：** 流程终点（可设置多个）
- **工序节点：** 表示生产工序
- **判断节点：** 条件分支判断
- **数据节点：** 数据记录/获取
- **通知节点：** 消息通知

### 8.2 流程类别管理
**功能路径：** 工艺设计 → 流程类别

**操作步骤：**
1. 点击"流程类别"进入类别管理页面
2. **新增类别：** 点击"新增" → 填写类别编码、类别名称 → 点击"保存"
3. **管理类别：** 对流程进行分类管理，便于检索

## 九、订单管理模块

### 9.1 生产订单管理
**功能路径：** 订单管理 → 生产订单

**操作步骤：**
1. 点击"生产订单"进入订单列表页面
2. **创建订单：** 点击"新增" → 填写订单编号、客户名称、产品型号、数量、交货日期等 → 点击"保存"
3. **订单评审：** 对订单进行技术评审和产能评审
4. **订单下达：** 评审通过后点击"下达"将订单投入生产

### 9.2 MRP运算
**功能路径：** 订单管理 → MRP运算

**操作步骤：**
1. 点击"MRP运算"进入MRP页面
2. **选择订单：** 勾选需要进行MRP运算的生产订单
3. **执行运算：** 点击"执行MRP"按钮 → 系统自动计算物料需求
4. **查看结果：** 查看物料需求清单，包括需求数量、需求时间、库存缺口等
5. **生成采购计划：** 根据MRP结果生成相应的采购申请

## 十二、常见操作说明

### 12.1 查询操作
- **精确查询：** 在查询框中输入完整关键字
- **模糊查询：** 输入部分关键字进行模糊匹配
- **组合查询：** 使用多个查询条件进行组合查询
- **重置查询：** 点击"重置"按钮清除查询条件

### 12.2 批量操作
- **全选：** 点击列表表头的复选框
- **多选：** 按住Ctrl键点击多个目标
- **批量删除：** 勾选多个目标 → 点击"批量删除" → 确认删除
- **批量审核：** 勾选多个目标 → 点击"批量审核" → 填写审核意见 → 确认

### 12.3 导出操作
1. 在列表页面完成数据筛选
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV/PDF）
4. 点击"确定"下载导出文件

### 12.4 打印操作
1. 在需要打印的页面点击"打印"按钮
2. 选择打印机
3. 设置打印份数和页面范围
4. 点击"打印"

## 十三、注意事项

### 13.1 数据安全
- 及时退出登录，防止他人冒用账号
- 重要数据操作前建议先备份
- 删除数据前确认数据是否还有使用价值

### 13.2 操作规范
- 按流程规范操作，避免数据错误
- 关键操作（如删除、审核）需谨慎确认
- 遇到问题及时联系系统管理员

### 13.3 性能维护
- 定期清理浏览器缓存
- 避免同时打开过多页面
- 大数据量操作建议错峰进行
